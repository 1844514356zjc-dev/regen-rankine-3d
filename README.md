# 一级回热朗肯循环 · 3D 动力装置仿真

单页交互式网页:中间是动态 3D 动力装置模型,左侧调参,右侧实时显示循环性能 + **T-s / P-v 图**。
水蒸气物性采用 **IAPWS-IF97** 工业标准(已本地 vendored,离线可用)。

## 运行

```bash
cd regen-rankine-3d
node serve.js          # 或 npm run serve
# 浏览器打开 http://localhost:8765/
```

> 需 Node ≥ 18。整站离线自包含(three.js 与 if97 均已 vendor 到 `vendor/`,不依赖任何 CDN)。
> 也可用 `python3 -m http.server 8765`,但 `serve.js` 会给 `.js` 返回正确的 `text/javascript` MIME,ES 模块更稳。

## 功能

- **左侧 — 参数控制**:锅炉压力、汽轮机入口温度、冷凝器压力、抽汽压力、汽轮机/泵等熵效率、蒸汽流量;4 个工况预设。越界参数自动夹紧并提示。
- **中间 — 3D 模型**:锅炉(炉膛辉光)、高压/低压汽轮机(叶片旋转)、发电机、回热加热器、凝汽器、两台给水泵;管道按工质温度着色(冷蓝→热红),蒸汽粒子沿管道流动,流量联动转速。鼠标旋转/滚轮缩放,可自动旋转。
- **右侧 — 输出**:热效率 η、机组功率、吸热 qᵢₙ、放热 qₒᵤₜ、净功/汽轮机功/泵功、抽汽率 y;回热 vs 无回热 vs 卡诺效率对比;7 个状态点参数表;T-s 图(橙=吸热/蓝=放热阴影);P-v 图(对数坐标)。

## 热力模型(开式/混合式给水加热器,单级抽汽)

状态点:`1` 冷凝器出口(饱液)→ 泵① → `2` → `3` 加热器出口(饱液)→ 泵② → `4` → `5` 锅炉出口(过热)→〔抽汽 `6`→加热器〕→ `7` 排汽 → 冷凝器。

- 抽汽率:`y = (h₃ − h₂)/(h₆ − h₂)`
- 汽轮机功:`wₜ = (h₅−h₆) + (1−y)(h₆−h₇)`;泵功:`wₚ = (1−y)(h₂−h₁) + (h₄−h₃)`
- 吸热 `qᵢₙ = h₅−h₄`;放热 `qₒᵤₜ = (1−y)(h₇−h₁)`;热效率 `η = (wₜ−wₚ)/qᵢₙ`
- 实际过程:泵 `h₂ = h₁ + (h₂ₛ−h₁)/ηₚ`;汽轮机 `h₆ = h₅ − ηₜ(h₅−h₆ₛ)`(等熵基准由 IF97 反求)

等熵效率默认 100%(理想);拖到实际值即可看到效率下降。
